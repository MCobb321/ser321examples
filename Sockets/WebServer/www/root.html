You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /multiply?num1=?&num2=? -- multiplies 2 numbers together
  • /github?query=[github api repo] -- returns github repo information
  • /reverse?str1=string1&str2=string2 -- reverses 2 strings
  • /larger?num1=?&num2=? -- determines if numbers are larger, small or equal
